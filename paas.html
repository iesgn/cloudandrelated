PaaS

PaaS: Platform as a Service, is a cloud computing model that allows the users to develop, deploy, run and manage applications witout taking care about the underlying layers

Source: https://azure.microsoft.com/es-es/overview/what-is-paas/
Source: https://medium.com/@lopezlucas/introducci%C3%B3n-a-cloud-saas-iaas-paas-772c92c29bb1
  • Developers only focused on developing applications
  • Developers don't care about the underlying layer (infrastructure), saving time and resources
  • PaaS can run anywhere: IaaS, physical machines or even containers.
  • Cloud Computing essential characteristics: elasticity, pay per use, ...
  • Big players (PaaS integrated in a full stack)
    • Amazon Web Service
    • Microsoft Azure
    • Google App Engine
    • Oracle Cloud Platform
  • Focused on deployment
    • Heroku
    • OpenShift
