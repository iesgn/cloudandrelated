Objectives

  • To get an overview of cloud computing
  • To understand the different layers and areas of application
  • To know the "forced" change of methodologies:
    • Infrastructure as Code
    • DevOps
  • To know the main related technologies:
    • Version Control Systems
    • Resource provisioning
    • Configuration Management
  • To learn through the practice: IaaS and PaaS
