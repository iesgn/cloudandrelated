Linux won

  • Innovations on almost every IT field are performed using Linux as OS and open source software
  • Cloud computing is paradigmatic: "Clouds are made of linux servers"
  • Windows won on desktop computers, but who cares today?
  • All the cloud is running on linux servers, but ... in cloud computing you don't care about the OS
  • Linux won :), but did we win? :(

It's good to get all the benefits of cloud computing technologies to feel comfortable using a linux box from the command line, because it's the natural environment to them
